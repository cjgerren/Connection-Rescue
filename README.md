# Connection Rescue

Connection Rescue is a real-time flight disruption assistance platform that helps travelers recover from missed or delayed flights.

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js + Express
- Database: Supabase
- Payments: Stripe

## Setup

### Frontend
cd frontend
npm install
npm run dev

### Backend
cd backend
npm install
npm run dev

## Environment Variables
See `.env` files in frontend and backend.

## Features
- Flight search & recovery
- Payment processing
- User authentication
